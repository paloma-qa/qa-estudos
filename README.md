# 🧪 Projeto de Testes — QA com JavaScript e API

## 📌 Sobre o projeto

Este repositório contém testes desenvolvidos com foco em validação de regras de negócio, utilizando JavaScript e testes de API.

O objetivo é simular cenários reais de QA, incluindo validações, identificação de bugs e organização de casos de teste.

---

## 🧪 Tipos de testes aplicados

### ✔️ Testes de lógica (JavaScript)

* Validação de usuário
* Validação de senha
* Cálculo de frete
* Regras de negócio (idade, desconto, etc.)

### ✔️ Testes de API

* GET (listagem e busca por ID)
* POST (criação de usuário)
* PUT (atualização)
* DELETE (remoção)
* Validação de status code
* Validação de estrutura de resposta

## 🛠️ Tecnologias utilizadas

* JavaScript

---

## ▶️ Como executar

1. Instale o Node.js
2. Clone o repositório
3. Execute no terminal:

```
node dia5.js
```

4. Para testes de API:

* Importar a coleção no Postman
