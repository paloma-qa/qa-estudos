🧪 Validação de Dados — JavaScript & QA

Testes e validações de dados com JavaScript e Postman — casos de teste, funções e cenários de erro.

## 📚 Conteúdo

| Arquivo | Conteúdo |
|---|---|
| dia1.js | Variáveis, tipos e função de login |
| dia2.js | Validação de idade e cálculo de desconto |
| dia3.js | Arrays, loops e validador de usuários |
| dia4.js | Validação de senha e cálculo de frete |
| dia5.js | Sistema de validação refatorado |
| postman-testes-api | Coleção Postman com 14 testes de API (GET, POST, PUT, DELETE) |

## 🛠️ Tecnologias
- JavaScript
- Node.js
- Postman
- Git & GitHub

## ▶️ Como rodar
1. Instale o Node.js em [nodejs.org](https://nodejs.org)
2. Clone o repositório
3. No terminal: `node dia5.js`
4. Para testes de API: importe o arquivo `postman-testes-api` no Postman
